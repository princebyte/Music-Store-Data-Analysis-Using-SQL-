# 🎵 SQL Project: Music Store Data Analysis

## 📌 Project Description

SQL project to analyze online music store data using PostgreSQL.
This project helps in understanding customer behavior, sales trends, and music preferences.

---

## 🎯 Objective

* Analyze sales data
* Identify top customers
* Find popular genres
* Understand business growth
